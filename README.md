IMDb Movies & Directors Analysis using Python and SQL

This project performs data analysis on an IMDb movie database using Python, Pandas, SQLite, and SQL queries.
The dataset contains information about movies and directors, including movie popularity, revenue, budget, ratings, and director details.

The project demonstrates how to:

Connect Python with a SQL database
Execute SQL queries using cursor objects
Fetch and analyze movie data
Convert SQL results into Pandas DataFrames
Perform data exploration and analytical queries
Features
Retrieve movie and director data from database tables
Analyze total number of movies in IMDb dataset
Search directors by specific names or patterns
Count female directors
Identify popular and bankable movies
Find highest-rated movies after 2000
Perform JOIN operations between movies and directors tables
Discover directors with maximum movies and profits

Learning Outcomes

Through this project, you can learn:

SQL querying fundamentals
Database handling in Python
Data analysis using Pandas
SQL JOIN, GROUP BY, ORDER BY, LIMIT, and aggregate functions
Converting raw SQL output into structured tabular format
Conclusion

This project is a beginner-friendly demonstration of combining SQL and Python for real-world data analysis. It helps in understanding how databases can be queried efficiently and analyzed using Pandas for meaningful insights from movie datasets.
